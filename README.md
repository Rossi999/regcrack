# regcrack - 嵌入式暫存器位元拆解工具

這是一個專為韌體工程師與嵌入式系統開發者設計的輕量級 C 語言工具。它可以快速將十六進位（Hex）的暫存器數值轉換為二進位波形，並逐一拆解 Bit 0 到 Bit 7 的電位狀態（HIGH/LOW），方便在 Debug 或對照 Datasheet 時使用。

---

## 🛠️ 功能特點
* **十六進位自動轉換：** 輸入 `0x5A` 等 Hex 值，自動解析。
* **視覺化二進位輸出：** 自動以每 4-bit 留空的方式顯示波形（例如 `0101 1010`）。
* **Bit 狀態列表：** 清晰列出 Bit [0] ~ Bit [7] 是 `★ HIGH (1)` 還是 `LOW (0)`。

---
